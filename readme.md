# Your Adidas Team App

This app is build with Stencil, a compiler for building fast web apps using Web Components.

## Getting Started

To start the project:

install it:

```bash
npm i
```

build the app for production, run:

```bash
npm run build
```

and run:

```bash
npm start
```

To run the unit tests once, run:

```
npm test
```
